# איך להפעיל את מערכת ניהול הישיבה
# How to Run the Yeshiva Management System

## 📋 יש לך 2 אפשרויות להפעלה:

---

## אפשרות 1: הפעלה מלאה (עם חלון Desktop) - מומלץ! ⭐

### השתמש בקובץ: `run_app.py`

קובץ זה מפעיל:
- ✅ שרת Flask (Backend)
- ✅ חלון Desktop (אם pywebview מותקן)
- ✅ פותח אוטומטית את האפליקציה

### איך להפעיל:

#### שיטה 1: לחיצה כפולה
פשוט **לחץ פעמיים** על הקובץ:
```
run_app.py
```

#### שיטה 2: מהטרמינל
```bash
cd "C:\Users\שלום\Downloads\ישיבה_חדש"
python run_app.py
```

### מה זה עושה:
1. **מפעיל את Flask** בפורט 5000
2. **מחכה 2 שניות** שהשרת יעלה
3. **פותח חלון Desktop** עם האפליקציה (אם יש pywebview)
4. **אם אין pywebview** - רק מפעיל את Flask ומדפיס:
   ```
   Open browser to: http://127.0.0.1:5000
   ```

---

## אפשרות 2: הפעלה רגילה (דפדפן בלבד)

### השתמש בקובץ: `app.py`

קובץ זה מפעיל רק את שרת Flask.

### איך להפעיל:

```bash
cd "C:\Users\שלום\Downloads\ישיבה_חדש"
python app.py
```

### מה זה עושה:
1. **מפעיל את Flask** בפורט 5000
2. **לא פותח חלון אוטומטית**
3. צריך **לפתוח דפדפן ידנית** ולגשת ל:
   ```
   http://127.0.0.1:5000
   ```

### מתי להשתמש באפשרות זו?
- כשרוצים **debug mode** (רואים שגיאות מפורטות)
- כשרוצים **auto-reload** (כל שינוי בקוד מאתחל את השרת)
- כשרוצים לעבוד עם **כמה דפדפנים בו-זמנית**

## 🔧 אם יש בעיות:

### בעיה: "ModuleNotFoundError"
**פתרון**: התקן את הספריות הנדרשות:
```bash
pip install -r requirements.txt
```

### בעיה: "Port 5000 already in use"
**פתרון**: יש שרת אחר שרץ על פורט 5000. סגור אותו או שנה את הפורט ב-`run_app.py`:
```python
app.run(debug=False, host='127.0.0.1', port=5001, use_reloader=False)
```

### בעיה: PyWebView לא פותח חלון
**זה תקין!** המערכת תדפיס:
```
Open browser to: http://127.0.0.1:5000
```
פשוט פתח את הדפדפן וגש לכתובת הזו.
